<h2>That's a simple project to automatically convert any temperature value.</h2>
<p>I made this project, just for study in Python.<p>
<br><br>

<p>
Formula Types:

K -> C: C = K - 273;<br>
K -> F: (K-273) * 1,8 + 32;<br>
C -> K: K = C + 273;<br>
C -> F: F = C * 1,8 + 32;<br>
F -> C: F = (F-32) / 1,8;<br>
F -> K: K = (F-32) * 5/9 + 273;<br>
</p>
